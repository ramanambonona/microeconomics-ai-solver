Getting Started

API Keys

MicroSolver AI requires you to provide your own API keys for AI services. This ensures complete privacy and control over your usage.

DeepSeek API (Recommended - Free)

  1. Visit DeepSeek Platform
  2. Create an account or sign in
  3. Generate a new API key
  4. Copy and save the key in the configuration section

OpenAI API (Paid)

  1. Visit OpenAI Platform
  2. Sign in to your account
  3. Create a new API key or use existing one
  4. Ensure you have sufficient credits

Problem Types

Consumer Theory

Solve utility maximization problems with budget constraints:

  • Utility Function: U(x,y) = x^α * y^(1-α)
  • Budget Constraint: p_x*x + p_y*y = I

Producer Theory

Solve cost minimization and profit maximization problems:

  • Production Function: Q = L^α * K^(1-α)
  • Cost Constraint: C = wL + rK

Market Equilibrium

Find market equilibrium prices and quantities:

  • Demand Function: Q_d = a - bP
  • Supply Function: Q_s = c + dP

Game Theory

Analyze strategic interactions and find Nash equilibria:

  • Payoff Matrix: [[(3,3),(0,5)],[(5,0),(1,1)]]

Input Format

Use standard mathematical notation:

Operation Notation Example
Exponent ^ x^2, y^(1-alpha)
Multiplication * p_x * x
Division / I / p_x
Greek Letters alpha, beta, gamma x^alpha * y^beta

Privacy & Security

  • API keys are stored locally in your browser
  • No user data is stored on our servers

Troubleshooting

Common Error Messages

  • Invalid API Key: Check your API key format and validity
  • Quota Exceeded: Your API provider has usage limits
  • Network Error: Check your internet connection
  • Backend Unavailable: Our server may be temporarily down

Performance Tips

  • Use DeepSeek API for free access with generous limits
  • Keep problem descriptions concise
  • Use standard mathematical notation
  • Check your API provider's status page for outages
