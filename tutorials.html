Quick Start Guide

Step 1: Get Your API Key

Visit DeepSeek Platform to get a free API key, or use your existing OpenAI API key.

Step 2: Configure the Solver

Enter your API key in the configuration section. Your key is stored locally and never sent to our servers.

Step 3: Enter Your Problem

Choose the problem type and enter your functions using mathematical notation. You can also add a description in natural language.

Step 4: Get Step-by-Step Solution

The AI will generate a detailed solution with LaTeX-formatted mathematical expressions that you can copy or download.
